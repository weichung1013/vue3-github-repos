## Project Setup

```sh

```

### Compile and Hot-Reload for Development

```sh
npm run serve:dev
```

### Compile and Minify for Production

```sh
npm run serve:prod
```

### Run Unit Tests with [Vitest](https://vitest.dev/)

```sh
npm run test:unit
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
